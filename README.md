# cloud-referee-kiro

Cloud Referee application for evaluating and providing guidance on cloud-related decisions.

## Project Structure

```
cloud-referee-kiro/
├── app.py              # Main application entry point
├── referee.py          # Referee module
├── prompts/
│   └── referee_prompt.txt  # Referee prompt template
├── .kiro/              # Kiro configuration directory
└── README.md           # Project documentation
```

## Usage

Run the application:

```bash
python app.py
```
